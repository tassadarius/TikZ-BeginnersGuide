# 06 Libraries

TikZ provides a good number of libraries which makes your life easier or adds new
features.

Some of them we actually have already used, e.g. `calc`, `positioning` or `shapes`.
However, there are many more:

## Automata

Draw finite automata and turing machines  
⇒ [link](https://tikz.dev/library-automata)

## Tree

(self-explanatory)  
⇒ [link](https://tikz.dev/library-trees)

## Matrix

Draw matrices more conveniently  
⇒ [link](https://tikz.dev/library-matrix)

## Circuit

Draw electric circuits  
⇒ [link](https://tikz.dev/library-circuits)


## Spy

Magnify a part of a picture  
⇒ [link](https://tikz.dev/library-spy)


## Patterns

Draw patterns, e.g. bricks or stripes  
⇒ [link](https://tikz.dev/library-patterns)
